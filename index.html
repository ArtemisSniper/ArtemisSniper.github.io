Artemis

A next-generation Minecraft username sniper.

Compatible

Supports all account types (Mojang, Microsoft, gift cards).

Auto Delay

Calculates an estimated delay from the name change endpoint.

Multi-Account

You can use multiple Minecraft accounts at once.

Efficient

Combines threading + sockets per request for swift speeds.

Auto Auth

Artemis automatically scans and authenticates all your accounts.

Easy to Use

Just put your accounts in accs.txt and start sniping!

System Requirements

Artemis is compatible with Windows, Linux and Mac. Requires Python 3.8 or above.

How can I calculate my delay?

Using the Auto Delay feature can give you a rough estimate of your offset, which can then be further tuned. Increasing your offset will send the requests earlier.
